# Diplom

Сайт для интерактивной 3D-визуализации учебного оборудования.

## Что реализовано

- выбор специальности: ОИБ, ПД, ЗЕМ;
- каталог учебного оборудования для каждой специальности;
- интерактивный просмотр 3D-модели в браузере;
- QR-код для открытия выбранной карточки оборудования;
- статическая структура без сборки и серверных зависимостей.

## Запуск на Linux

```bash
python3 -m http.server 8080
```

После запуска откройте:

```text
http://localhost:8080
```

Для работы 3D-моделей и QR-кодов нужен доступ к интернету, потому что демо
использует открытые GLB-модели и внешний сервис генерации QR.
