# Diplom

Сайт для интерактивной 3D-визуализации учебного оборудования.

## Что реализовано

- выбор специальности: ОИБ, ПД, ЗЕМ;
- каталог учебного оборудования для каждой специальности;
- интерактивный просмотр 3D-модели в браузере;
- QR-код для открытия выбранной карточки оборудования на домене учреждения;
- статическая структура без сборки и серверных зависимостей.

## Локальный запуск на Linux

```bash
python3 -m http.server 8080
```

После запуска откройте:

```text
http://localhost:8080
```

## Размещение на сервере учебного заведения

Проект статический, поэтому его можно разместить на Apache, Nginx или любом
другом HTTP-сервере. Достаточно скопировать файлы `index.html`, `styles.css`,
`app.js` и папку `vendor/` в директорию сайта, например:

```bash
sudo mkdir -p /var/www/equipment-3d
sudo cp -r index.html styles.css app.js vendor /var/www/equipment-3d/
```

Пример Nginx-конфигурации:

```nginx
server {
    listen 80;
    server_name equipment.college.local;
    root /var/www/equipment-3d;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

QR-коды генерируются локально в браузере и ведут на текущий домен сайта. Если
сайт открыт как `https://equipment.college.ru/`, QR-ссылка будет использовать
этот адрес.

Локальная CSS-3D-визуализация работает без интернета. Ссылки на GLB-модели в
`app.js` можно заменить на файлы учреждения, например `models/camera.glb`.
